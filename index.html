Inline

Inline styles specifically affect the section they are written in, its best to use inline styles if the code only is being used once, if not its best to use external styles. Inline example below

Style is an example of an inline style, it specifically affects that chunk of code rather than all the code. In this example it will make this specific image 200 pixels in height. This allows CSS to be inside of the HTML tag.

Internal

Internal styles consist of the <style> and <head> tags. This is better if one web page needs to use specific code otherwise its best to use external so if i need to do any changes i only have to do it once rather than going to each page with the code.

The <head> tag contains everything that has to be read by the browser first as it tells the browser what's ahead.

The <style> tag contains CSS which is applied to the content in the body of the program.
